# MeihoUnderSeatStorageHobieProAngler-
Under Seat Storage System for Hobie Pro Angler optimized for Meiho Boxes. Made out of polycarbonate.  
# Meiho Box Configurations
* Configuration #1
   * 1x 3020 NDDM
   * 2x 3020 ND
   * 2x 3020 NS
   * 2x 1010W
* Configuration #2
   * 1x 3020 NDDM
   * 2x 3020 ND
   * 4x 3010 NS
   * 2x Reversible D-86
* Configuration #3
   * 1x 3020 NDDM
   * 4x 3010 ND
   * 4x 3010 NS
   * 2x Reversible D-86
